# Unofficial Cure Cheat Sheet

A web tool for efficiently identifying conditions and treatments in *Cure - A Hospital Simulator* based on symptom selection.

---

## Purpose

This project is designed as a fast reference tool for in-game use. It helps match symptoms to possible conditions and view relevant treatments without manually diagnosing with tools in-game.

It is intended for convenience and learning, not as an authoritative medical or game source.

---

## Data

The matching system is based on structured in-game observations and community testing.
